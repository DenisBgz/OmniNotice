# OmniNotice

Сервис для рассылок с использованием ИИ.



## Стек

- React 18 + TypeScript
- Vite
- Chakra UI

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Переменные окружения

Скопируйте `.env.example` в `.env` и при необходимости измените переменные.

Synora, Oracle являются внутренними микросервисами

Synora - для отправки писем по email
Oracle - обработка текста с помощью ИИ.
